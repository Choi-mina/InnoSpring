# InnoSpring
## 🖥️ 프로젝트 소개
아티스트와 팬이 소통할 수 있는 웹 플랫폼<br/>
(기존에 있는 플랫폼에 불편한 점을 개선)

<br/>

### 주요 기능 🗃️
Security를 Artist/Manager/Fan 역할별로 기능 제한
#### Community
+ Fan, Artist 구분 없이 누구나 쓸 수 있는 게시판
#### Artist
+ Artist이 쓴 글을 볼 수 있는 게시판
#### Schedule
+ Artist의 스케줄을 확인할 수 있는 게시판
+ Manager, Artist만 작성 가능
####  Notice
+ 공지사항 게시판
+ Manager, Artist만 작성 가능
#### 마이페이지
+ 회원정보 수정
+ 내가 쓴 글, 댓글 조회
+ 회원 탈퇴

<br/>
